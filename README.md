# Event Invitation App

A small vanilla JavaScript web application for managing event invitations.

## Features
- Create event details (title, date, location, description)
- Add and remove participants dynamically
- Email validation for participants
- Form validation before submission
- Simulated email sending
- Dark Mode support with localStorage

## Technologies
- HTML
- CSS
- JavaScript (Vanilla)

## Purpose
This project was created as a practice project to improve JavaScript form handling,
DOM manipulation, validation logic, and basic UI features.

## Notes
Email sending is simulated and logged to the console.
